Getting Started

Size Guide for SWAY is a Shopify App utilizing only the Theme App Extension feature for the size guide to be transferrable.

Theme app extension only allows 100kb so uploading files and adding some code are necessary.

Installation

  • 1. From Shopify Partners account, under Apps menu open 'Size Guide by SWAY' app.
  • 2. Under Merchant install link, share the link provided to install app. This requires the app to be live. Being live requires a server.
  • 3. Before install, please follow shopify documentation to make the app live here to successfully run shopify app serve command.
  • 4. Paste installation link and install the app. It is good to go.

Requirements for it to work: Steps

  • 1. Uploading all Assets file here into assets folder of shopify
  • 2. Adding custom code into the theme.liquid file
  • 3. Adding the 'Size Guide by SWAY' block into the theme using the customizer
  • 4. Add shirt into the list
  • 5. Further changes on code depending on theme

Step 1: Upload all files from Assets folder here into assets folder in shopify

1. Download Assets file from here

2. Go to online store.

3. Open Assets folder. Click on "Add a new asset" Add all files.

3. You may need to change {{% if %}} statement in step 1 depending on the theme's product page template name.

Step 3: Adding the 'Size Guide by SWAY' block into the theme using the customizer

1. Go to online store. Under the theme you want to add 'Size Guide', click on 'Customize' option.

2. On the right side part of the screen, look for click on 'Add Section' option and under Apps click on 'Product Pages embedded through Size Guide by SWAY'.

3. Hide default product page of theme and drag 'Product Page' block next to 'Header' block or 'Breadcrumbs' block and Click 'Save' on the top.

Step 4: Add shirt into the list

1. Inside App section, click on 'Product Pages' and it will open customization settings.

2. Under "Add Products here for Size Guide", edit and add product name you want the size guide to be added. Spacing and capitalization should be followed when adding product name and each product name needs to be enclosed with apostrophe and are separated with comma.

3. Hit 'Save'.

Step 5: Further changes on code depending on theme.

1. If font might be different. Please check if theme style is overriding what is set on affected texts.

2. Letter spacing, text decoration, overflow's might need some override.

3. Parent html's height needs to be 100%.
